## Project Overview
This project analyzes customer churn behavior for a telecom company using Power BI.  
The objective is to identify key factors influencing customer churn and provide
data-driven insights to reduce churn and improve customer retention.

## Dataset
- Source: Telco Customer Churn dataset
- Records: Customer-level data
- Key features:
  - Demographics
  - Service subscriptions
  - Contract details
  - Payment methods
  - Churn status

## Business Questions
1. What is the overall customer churn rate?
2. Which contract types have the highest churn?
3. How does tenure impact churn behavior?
4. Which payment methods are associated with higher churn?
5. What services are most common among churned customers?

## KPIs
- Total Customers
- Churned Customers
- Churn Rate (%)
- Average Tenure
- Monthly Charges (Avg)

## Dashboard Features
- Churn overview metrics
- Churn by contract type
- Churn by payment method
- Tenure vs churn analysis
- Service usage comparison

## Tools Used
- Power BI (Data modeling, DAX, visualization)
- Excel (Data source)

## Output
An interactive Power BI dashboard providing actionable churn insights for business decision-making.
